# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the NextGenTra LMS project.

## What is an ADR?

An ADR documents an important architecture decision made during the development of the project. It includes:

- **Context**: What is the issue we're addressing?
- **Decision**: What decision have we made?
- **Consequences**: What are the positive and negative consequences?
